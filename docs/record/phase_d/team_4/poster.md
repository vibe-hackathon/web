# NightEcho AI
> Transforming the lives of the visually impaired with real-time audio guidance.

**Track:** T01 (Best Social Impact)

## Problem
The visually impaired face significant challenges in navigating urban environments safely and independently.

## Solution
NightEcho AI is an AI-driven navigation app that uses GPT-4 for audio generation and YOLOv8 for object detection, providing real-time audio guidance and enhancing independence and safety.

## Tech Stack
- GPT-4 for audio generation
- YOLOv8 for object detection
- React for frontend
- Flask for backend
- AWS S3 for storage

## Key Features
1. Real-time audio guidance for obstacles and directions
2. Haptic feedback for enhanced user interaction
3. Customizable settings for user preferences
