# Radiocure: Precision Radioisotope Therapy
> Revolutionizing cancer treatment with precision radioisotope therapy.

**Track:** T01

## Problem
Cancer treatments often cause significant damage to healthy tissues, leading to severe side effects. Current methods lack precision in targeting cancer cells.

## Solution
We aim to develop a new class of targeted cancer therapies using precision radioisotope therapy, minimizing damage to healthy tissues and improving patient outcomes.

## Tech Stack
- TensorFlow 2.x
- Kubernetes
- OpenVINO
- React for frontend
- Node.js for backend
- PostgreSQL for database
