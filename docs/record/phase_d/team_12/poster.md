# VoiceBridge
> Empowering Citizens for Real-Time Policy Influence

**Track:** T01

## Problem
Inadequate channels for citizen engagement and influence on policy-making processes.

## Solution
VoiceBridge uses AI-driven sentiment analysis, blockchain for transparency, and mobile-first design to enable citizens to directly influence policy decisions in real-time.

## Tech Stack
- TensorFlow
- Ethereum Blockchain
- React Native
