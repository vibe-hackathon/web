# EduGuru
> Personalized Learning Assistant for Every Student

**Track:** T01 (Best Social Impact)

## Problem
Visual learning gaps among students, especially the visually impaired, hinder educational outcomes.

## Solution
Develop a personalized learning assistant using AI to adapt to individual learning styles and needs, ensuring no student is left behind.

## Tech Stack
- GPT-4 API
- Firebase
- React
- Node.js

## Team
- A_01: PM/Visionary, defining scope, success metrics, and user stories
- A_33: Tech Lead, handling backend and integration
- A_31: Designer, ensuring UX/UI polish
- A_37: Coder, implementing features and coding

## Key Features
1. Adaptive learning paths based on student performance
2. Real-time feedback and adjustments
3. Accessibility features for visually impaired students

## Demo Pitch
Our demo will showcase a visually impaired student using the personalized learning assistant, adapting to their unique learning style and providing real-time feedback. This will highlight the effectiveness of our solution in addressing educational gaps.
