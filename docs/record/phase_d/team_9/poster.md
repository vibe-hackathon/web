# UrbanVox
> Transform urban walls into platforms for anonymous social commentary using AR.

**Track:** T01

## Problem
Urban spaces often lack a platform for anonymous, empathetic social commentary. This leads to a lack of community engagement and understanding.

## Solution
UrbanVox transforms urban walls into interactive platforms where users can leave anonymous comments using augmented reality. These comments can be viewed and responded to by others, fostering dialogue and empathy.

## Tech Stack
- React Native
- ARKit
- Firebase
- TensorFlow Lite

## Key Features
1. Onboarding: Simple and intuitive registration process.
2. AR Preview: Users can see their comments in augmented reality before submitting.
3. Comment Submission: Users can leave anonymous comments on any urban wall.
4. Comment Viewing: Users can view and respond to comments left by others.

## Demo Pitch
UrbanVox is an AR app that turns urban walls into interactive platforms for anonymous social commentary. Users can leave comments, view responses, and engage in meaningful conversations, fostering a sense of community and empathy.
