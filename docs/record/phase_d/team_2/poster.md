# ConnectBridge
> Real-time resource connection for the homeless

**Track:** T01

## Problem
Homelessness is a complex issue with many gaps in resource allocation. Current systems are inefficient and often fail to connect individuals with available resources.

## Solution
ConnectBridge uses AI to match homeless individuals with available resources in real-time, ensuring they receive the help they need. Our platform will leverage machine learning models to predict resource availability and provide personalized recommendations.

## Tech Stack
- TensorFlow
- Firebase Realtime Database
- GPT-4 API
- React.js
- Node.js
